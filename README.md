# lap2.
Problem Statement: The objective of this project is to classify the Color of Arabica coffee beans based on their sensory evaluation scores (such as Aroma, Flavor, and Acidity) and physical defects data.
Machine Learning Type: Classification, as the target variable (Color) consists of discrete categories.
Target Variable: Color.
Goal: To automate the quality assessment process by identifying how the taste profile of the coffee relates to the physical appearance of the beans.
